Learning Tailwind by building real pages

This website is part of my learning journey with Tailwind CSS. I built it step by step to understand how utility classes can help me create beautiful, responsive designs quickly and consistently.

Through hands-on practice, I’m improving my CSS skills and learning core concepts like spacing, layout, colors, borders, shadows, and responsive design — one page at a time.
