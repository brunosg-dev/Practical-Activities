About Module

In Module 4, I learned how to build well-structured and professional web pages using semantic HTML elements such as <header>, <main>, <article>, and <footer>, which give meaning and organization to the content. I also discovered how CSS controls layout and display by defining block-level and inline-level elements, using positioning properties like static, relative, and absolute, and applying Flexbox to make layouts responsive across devices. Additionally, I learned to use the <head> section effectively by adding meta tags, <link> for external stylesheets and fonts, and <script> for JavaScript interactivity. Overall, this module helped me understand how structure, style, and interactivity work together to create a complete and modern webpage.
