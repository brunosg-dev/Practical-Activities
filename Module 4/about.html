About Module

In Module 4, I learned how to structure my HTML documents using semantic elements, which give meaning to the content and help both developers and browsers understand how a page is organized. Instead of relying only on <div> tags, I now know how to use elements like <header>, <main>, <article>, <section>, and <footer> to define clear sections of a webpage. This makes my code more professional, readable, and SEO-friendly. For example, I can use <nav> for navigation links or <aside> for side content, which improves accessibility and provides a better experience for users and search engines.



I also learned how CSS controls the layout and display of HTML elements. Every element is either a block-level or inline-level element by default, and understanding their behavior helps me manage spacing and alignment properly. I now know how to use properties like position (static, relative, absolute, fixed, and sticky) to move elements around the page and how to apply Flexbox to create flexible, responsive layouts that adjust across devices. This knowledge made me realize how structure and design work together — HTML gives meaning to content, while CSS defines how that content appears visually.



Finally, I learned how to enhance a webpage with elements inside the <head> section and other professional features. I discovered how to use meta tags like <meta name="description"> and <meta name="keywords"> to define descriptions, keywords, and authorship for SEO and responsiveness. I also learned how to link external stylesheets and libraries using the <link> tag, import fonts from Google Fonts, and add scalable icons from Font Awesome. Lastly, I explored how to use the <script> tag to add JavaScript for interactivity. I practiced creating a complete webpage with a <header>, <main> section, and <footer>, applying everything I learned semantically and visually. This module helped me understand not just how to build a webpage, but how to build it the right way — organized, meaningful, and ready for real-world development.
